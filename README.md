# RISEN Knowledge OS

## Mission

**理解から、価値をつくる。**

RISENは、現場の出来事を単なる記録として保存するのではなく、
Knowledge（知識）として育てるためのプラットフォームです。

入力方法（MySQL・CSV・Web・音声など）に依存せず、
すべての情報をEventとして理解し、
Knowledgeへ変換し、
AI・検索・分析・文書生成へ活用します。

---

## Philosophy

**「記録を残す」のではなく、「知識を育てる。」**
