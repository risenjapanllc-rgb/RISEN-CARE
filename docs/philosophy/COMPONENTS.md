# RISEN CARE Components

Version 1.0

---

# Purpose

Components are the building blocks of RISEN CARE.

Every component exists to deepen understanding.

Screens are composed of reusable components.

Components are never designed for appearance alone.

Each component has a clear purpose.

---

# Component Principles

Every component should be:

* Simple
* Reusable
* Explainable
* Consistent
* Human-centered

If a component does not improve understanding, it should not exist.

# Component Rules

Components must be reusable.

Components must support Auto Layout.

Components should avoid fixed widths whenever possible.

Each component should have one clear responsibility.

Avoid combining multiple purposes into one component.

# Component Checklist

Before creating a component, ask:

Does it deepen understanding?

Can it be reused?

Does it have only one responsibility?

Can users understand it immediately?

Is supporting evidence accessible?

If not, redesign it.

---

# Closing Statement

Components are not visual elements.

Components are units of understanding.

Every component should help care professionals better understand the person they support.
